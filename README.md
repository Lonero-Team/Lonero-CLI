# Usage
<!-- usage -->
```sh-session
$ npm install -g lonero-cli
$ lonero-cli COMMAND
running command...
$ lonero-cli (-v|--version|version)
lonero-cli/0.0.4 linux-x64 node-v12.4.0
$ lonero-cli --help [COMMAND]
USAGE
  $ lonero-cli COMMAND
...
```
